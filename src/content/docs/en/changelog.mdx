# Changelog

## [3.0.2] - 2025-05-22
### 🐛 Bug Fixes
- Fixed behavior of empty separators
  - Now, when `data-counterjs-time-separator=""` is set, the time separator won't be displayed
  - The same applies to `data-counterjs-date-separator` and `data-counterjs-datetime-separator`
  - Default separator values changed to empty (`''`) instead of spaces or colons

### 🛠 Improvements
- Better handling of empty values for separators
- More consistent code in separator rendering

### How to Use
To remove separators:
```html
<div data-counterjs
     data-counterjs-date="31/12/2025 23:59:59"
     data-counterjs-time-separator=""
     data-counterjs-date-separator=""
     data-counterjs-datetime-separator="">
    Loading...
</div>
```

## [3.0.0] - 2025-05-22
### ✨ New Features
- Added support for date and time in DD/MM/YYYY HH:MM:SS format
- Customizable separators:
  - `dateSeparator`: to separate date items (year/month/day)
  - `timeSeparator`: to separate time items (hour:minute:second)
  - `dateTimeSeparator`: to separate date from time
- New `hideWhenZero` option to hide counter when it reaches zero

### 🛠 Improvements
- Enhanced counter display logic
- Better error handling for invalid dates
- Optimized and cleaner code
