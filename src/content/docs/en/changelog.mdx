# Changelog

## [3.0.1] - 2025-05-22
### 🌐 Internationalization
- Translated all content to English
- Updated examples to use English text
- Added comprehensive README.md in English
- Improved documentation with JSDoc
- Better code organization

## [3.0.0] - 2025-05-22
### ✨ New Features
- Added support for date and time in DD/MM/YYYY HH:MM:SS format
- Customizable separators:
  - `dateSeparator`: to separate date items (year/month/day)
  - `timeSeparator`: to separate time items (hour:minute:second)
  - `dateTimeSeparator`: to separate date from time
- New `hideWhenZero` option to hide counter when it reaches zero

### 🛠 Improvements
- Enhanced counter display logic
- Better error handling for invalid dates
- Optimized and cleaner code

## [2.1.1] - 2025-05-22
### 📝 Documentation
- Updated example to use CDN

## [2.0.1] - Project Moved to GitHub
- Initial CounterJS version
- Basic countdown functionality
- Support for custom formatting
- Simple HTML attributes integration

## [1.0.0] - Initial Release
- Previously hosted on Gist
