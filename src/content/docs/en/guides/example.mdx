# Practical Examples

Here are some examples of how you can use CounterJS in your projects.

## Example 1: New Year Countdown

```html
<div class="countdown-container">
    <h2>New Year Countdown</h2>
    <div data-counterjs data-counterjs-date="31/12/2025 23:59:59">
        Loading countdown...
    </div>
</div>

<style>
.countdown-container {
    text-align: center;
    font-family: Arial, sans-serif;
    padding: 20px;
    border-radius: 8px;
    background-color: #f5f5f5;
    max-width: 500px;
    margin: 0 auto;
}

.countdown-container h2 {
    color: #333;
    margin-bottom: 20px;
}

[data-counterjs] {
    font-size: 24px;
    font-weight: bold;
    color: #2c3e50;
}
</style>

<script src="https://cdn.jsdelivr.net/gh/eulucastiagolt/counterjs@3.0.1/counter.min.js"></script>
<script>
    document.addEventListener('DOMContentLoaded', function() {
        const counter = new CounterJS();
        counter.init();
    });
</script>
```

## Example 2: Product Launch Countdown

```html
<div class="launch-countdown">
    <h2>Coming Soon!</h2>
    <p>Our new product will be launched in:</p>
    <div data-counterjs 
         data-counterjs-date="07/01/2025 10:00:00"
         data-counterjs-hide-year="true"
         data-counterjs-hide-second="false">
        Loading countdown...
    </div>
</div>

<style>
.launch-countdown {
    text-align: center;
    padding: 30px;
    background: linear-gradient(135deg, #6e8efb, #a777e3);
    color: white;
    border-radius: 10px;
    max-width: 600px;
    margin: 20px auto;
    box-shadow: 0 4px 6px rgba(0, 0, 0, 0.1);
}

.launch-countdown h2 {
    font-size: 2em;
    margin-bottom: 10px;
}

.launch-countdown p {
    font-size: 1.2em;
    margin-bottom: 20px;
}

[data-counterjs] {
    font-size: 1.8em;
    font-weight: bold;
    letter-spacing: 2px;
}
</style>

<script src="https://cdn.jsdelivr.net/gh/eulucastiagolt/counterjs@3.0.1/counter.min.js"></script>
<script>
    document.addEventListener('DOMContentLoaded', function() {
        const counter = new CounterJS();
        counter.init();
    });
</script>
```
