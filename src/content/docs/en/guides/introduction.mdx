# Introduction to CounterJS

Welcome to CounterJS! This library is designed to simplify the creation and management of countdown timers in your web applications.

## What is CounterJS?

CounterJS is a lightweight library that makes it easy to create customizable countdown timers. With it, you can:

- Display countdowns to future dates and times
- Customize which time units are displayed (years, months, days, hours, minutes, seconds)
- Adjust separators between time units
- Automatically hide the counter when the target date is reached

## Key Features

- **Easy to use**: Just add a few attributes to your HTML
- **Lightweight**: Only ~3KB minified
- **Customizable**: Adapt the appearance to your design
- **Responsive**: Works on all screen sizes
- **Browser Support**: Compatible with all modern browsers and IE11+
