# Welcome to CounterJS

CounterJS is a lightweight and easy-to-use JavaScript library for creating customizable countdown timers in your web applications. With support for multiple time units and customization options, you can create beautiful countdowns for any occasion.

## Get Started

To start using CounterJS, follow these simple steps:

1. [Install CounterJS](/en/guides/installation/)
2. Follow the [Getting Started guide](/en/guides/getting-started/)
3. Check out the [Examples](/en/guides/example/)

## Key Features

- **Easy to use**: Add countdowns with just a few lines of code
- **Lightweight**: Only ~3KB minified
- **Customizable**: Control which time units to display and how to format them
- **Responsive**: Works great on mobile and desktop
- **No dependencies**: Works with vanilla JavaScript

## Quick Example

```html
<div data-counterjs data-counterjs-date="31/12/2025 23:59:59">
  Loading countdown...
</div>

<script src="https://cdn.jsdelivr.net/gh/eulucastiagolt/counterjs@3.0.1/counter.min.js"></script>
<script>
  document.addEventListener('DOMContentLoaded', function() {
    const counter = new CounterJS();
    counter.init();
  });
</script>
```
