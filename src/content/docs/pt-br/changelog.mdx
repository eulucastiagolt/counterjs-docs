# Registro de Alterações

## [3.0.1] - 22/05/2025
### 🌐 Internacionalização
- Todo o conteúdo traduzido para inglês
- Exemplos atualizados para usar texto em inglês
- Adicionado README.md abrangente em inglês
- Documentação aprimorada com JSDoc
- Melhor organização do código

## [3.0.0] - 22/05/2025
### ✨ Novos Recursos
- Adicionado suporte para data e hora no formato DD/MM/AAAA HH:MM:SS
- Separadores personalizáveis:
  - `dateSeparator`: para separar itens de data (ano/mês/dia)
  - `timeSeparator`: para separar itens de hora (hora:minuto:segundo)
  - `dateTimeSeparator`: para separar data de hora
- Nova opção `hideWhenZero` para ocultar o contador quando chegar a zero

### 🛠 Melhorias
- Lógica de exibição do contador aprimorada
- Melhor tratamento de erros para datas inválidas
- Código otimizado e mais limpo

## [2.1.1] - 22/05/2025
### 📝 Documentação
- Exemplo atualizado para usar CDN

## [2.0.1] - Projeto movido para o GitHub
- Versão inicial do CounterJS
- Funcionalidade básica de contagem regressiva
- Suporte para formatação personalizada
- Integração simples com atributos HTML

## [1.0.0] - Versão Inicial
- Anteriormente hospedado no Gist

## Usando Versões Anteriores

```html
<!-- Para a versão 2.1.1 -->
<script src="https://cdn.jsdelivr.net/gh/eulucastiagolt/counterjs@2.1.1/counter.min.js"></script>

<!-- Para a versão 2.0.1 -->
<script src="https://cdn.jsdelivr.net/gh/eulucastiagolt/counterjs@2.0.1/counter.min.js"></script>
```
