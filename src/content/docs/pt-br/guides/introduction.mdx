# Introdução ao CounterJS

Bem-vindo ao CounterJS! Esta biblioteca foi projetada para simplificar a criação e gerenciamento de contadores regressivos em suas aplicações web.

## O que é o CounterJS?

O CounterJS é uma biblioteca leve que permite criar contadores regressivos personalizáveis ​​com facilidade. Com ela, você pode:

- Exibir contagens regressivas para datas e horários futuros
- Personalizar quais unidades de tempo são exibidas (anos, meses, dias, horas, minutos, segundos)
- Ajustar os separadores entre as unidades de tempo
- Ocultar automaticamente o contador quando a data alvo for atingida

## Principais Recursos

- **Fácil de usar**: Basta adicionar alguns atributos ao seu HTML
- **Leve**: Apenas ~3KB minificado
- **Personalizável**: Adapte a aparência ao seu design
- **Responsivo**: Funciona em todos os tamanhos de tela
- **Suporte a navegadores**: Compatível com todos os navegadores modernos e IE11+
