# Bem-vindo ao CounterJS

CounterJS é uma biblioteca JavaScript leve e fácil de usar para criar contadores regressivos personalizáveis em suas aplicações web. Com suporte a múltiplas unidades de tempo e opções de personalização, você pode criar contadores atraentes para qualquer ocasião.

## Começando

Para começar a usar o CounterJS, siga estes passos simples:

1. [Instale o CounterJS](/pt-br/guides/installation/)
2. Siga o [guia de primeiros passos](/pt-br/guides/getting-started/)
3. Confira os [exemplos práticos](/pt-br/guides/example/)

## Recursos Principais

- **Fácil de usar**: Adicione contadores com apenas algumas linhas de código
- **Leve**: Apenas ~3KB minificado
- **Personalizável**: Controle quais unidades de tempo exibir e como formatá-las
- **Responsivo**: Funciona perfeitamente em dispositivos móveis e desktop
- **Sem dependências**: Funciona com JavaScript puro

## Exemplo Rápido

```html
<div data-counterjs data-counterjs-date="31/12/2025 23:59:59">
  Carregando contador...
</div>

<script src="https://cdn.jsdelivr.net/gh/eulucastiagolt/counterjs@3.0.1/counter.min.js"></script>
<script>
  document.addEventListener('DOMContentLoaded', function() {
    const counter = new CounterJS();
    counter.init();
  });
</script>
```
